### 設定したこと
vim 入力
daily note作成
daily noteにhotkeyを設定

#### vim 入力
settings -> editor -> Vim key bindings -> on

#### daily note作成
settings -> plugins -> Daily notes -> on

#### daily noteにhotkeyを設定
settings -> hotkey -> input today -> on

### どうなるか
vim 入力できるようになって、ctrl + t で今日のメモ帳が開けるようになる
最高
